Professional projects

Here are the projects that I have made in a professional context.

Detection of logical models on energy data in order to optimize energy consumption. Each survey analysed had at least 30 000 records (the biggest one had 3M records). To give you an idea, analysing 30 000 records takes about 10 seconds, and around 5 minutes with 3M records.

Developping context
  • Developped from 0 by me,
  • I was managed by a team of 3 engineers in various energy domains.
Technologies used :
  • Python (programming language),
  • pandas (data analysis),
  • matplotlib (graphics),
  • Tkinter (interface),
  • SQLite (database)
  • EzAPI (progress bar) - my api ! ;)
Constraints :
  • Code has to respect PEP-8 conventions,
  • Code has to respect python conventions,
  • Code has to be commented,
  • Code has to be tested,
  • Code has to be documented (for the next dev, and the user),
  • No object oriented programming,
  • Code has to be maintainable by non-professional people.
All the initial objectives were reached, and I could do more than was expected. The software permits to the engineers to gain 2 days of works for every survey. Now they only need ONE day of work instead of 3 days before.
