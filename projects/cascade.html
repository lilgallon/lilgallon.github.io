Context

I made this project for my last year at "IUT Informatique" of Bordeaux to learn android development (2017/2018). It was meant to be a small project, but I decided to work hard to get a game that I could put on google play.


Installation

Step 1:

Download the apk file.

Step 2:

Connect your phone to your computer, then put the apk file on it.

Step 3:

Execute the .apk file from your phone, if it asks you to toggle a setting, toggle it. It is normal because the app is not on google play for the moment.

Step 4:

Beat your best records !


How to use it / rules

Objective

The goal is to break a wall of bricks by getting the best score possible. More you break bricks at the same time, more you will get points.

You can't destroy a single brick : the brick must have at least a same color neighbor.

Score calculation

Score = number of cells removed * number of cells removed * 5

When the game finishes, so when there is no more cell to remove, we remove (number of cells remaining * 50) to the score. If there is no cell remaining on the grid, the user gets a +500 score bonus.
