    How to install bettercolors
    1. Download Forge for your Minecraft version by clicking on Installer,
    2. Execute the installer by double-clicking, and select Client, and it should install automatically,
    3. Download bettercolors.jar and put it in your minecraft directory:
    3.1. Press windows key and R at the same time,
    3.2. Write %appdata%, press enter
    3.3. Go to .minecraft and create a folder mods
    3.4. Then put bettercolors.jar in it.
    4. Launch your game with Forge
    5. Once you are in the game, you can press INSERT key to toggle the interface
    If you fail, go to youtube, and search how to install a minecraft mod. It is easy. If you need more help about the mod, go to the github page right here.

    Music Genre Finder

    python â€¢ utility
    no release yet
    ðŸ”Ž It looks at music names, detects the genre, and creates playlists.

    why?

    Currently, all projects dealing with genre classification on Github are using AI (I guess that it's the trend) and only detect very basic genres like Jazz, Rock, and Pop.
    The goal of this project is to detect very specific genres like Dubstep, Rap, Lofi, Future Bass, and so on.

    how?

    To do so, we use the Youtube API with a dictionnary that we built. This dictionnary consists in giving scores for each genre according to the Youtube channel. For example, for UKF Dubstep, we associate a score of 100 for Dubstep. So if we found that the music has been uploaded to UKF Dubstep, we can be sure that it is a dubstep music.

    Cascade

    android â€¢ game
    version 2.0
    The goal of the game is to destroy a brick wall in a as less clicks as possible. But ... care to not let unbreakable bricks...

    context

    Made for the Introduction to mobile programming course at the Bordeaux Institute of Technology (grade: 19/20).

    rules

    • You can't destroy a single brick: the brick must have at least a same color neighbor,
    • The top 10 scores are visible on the scoreboard page,
    • For the score calculation, see above.
    Score calculation:
    score = (number of cells removed)^2 * 5
    When the game finishes, so when there is no more cell to remove, we remove number of cells remaining * 50 to the score. If there is no cell remaining on the grid, the user gets a +500 score bonus.
