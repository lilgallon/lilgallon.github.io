    PlaylistsByGenre

    no release
    üîé It looks at music names, detects the genre, and creates playlists.

    why?

    Currently, all projects dealing with genre classification on Github are using AI (I guess that it's the trend) and only detect very basic genres like Jazz, Rock, and Pop.
    The goal of this project is to detect very specific genres like Dubstep, Rap, Lofi, Future Bass, and so on.

    how?

    To do so, we use the Youtube API with a dictionnary that we built. This dictionnary consists in giving scores for each genre according to the Youtube channel. For example, for UKF Dubstep, we associate a score of 100 for Dubstep. So if we found that the music has been uploaded to UKF Dubstep, we can be sure that it is a dubstep music.
