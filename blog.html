EzAPI version 1.0 release

by N3ROO, 29/03/19



EzAPI is finally released. It is a compilation of useful and VERY simple python APIs. Those are made to be used in a non-object oriented programming context and can be used by beginners to do complex things in an easy way.
EzProgressBar API was ready since 2018, but I finished working on EzWebScraping today. I put a lot of work into the documentation, so that a beginner could use this API easily. Here is the link to the API. Even if you won't use it, take a look ! :)


Nero engine : Text rendering & Hud implementation

by N3ROO, 24/03/19

Implementing the text renderer was easier than expected. JOGL provides a simple way to render text using java.awt fonts. Here is a sample : 

                
                    // Init text renderer before
                    TextRenderer textRenderer = new TextRenderer(new Font("SansSerif", Font.PLAIN, 36));
                    // ...
                    textRenderer.beginRendering(
                            Renderer.getWindowWidth(),
                            Renderer.getWindowHeight()
                    );
                    textRenderer.setColor(red, green, blue, alpha);
                    textRenderer.draw(text, x + (int) xTranslation + xShift, y + (int) yTranslation + yShift);
                    textRenderer.endRendering();

However, it seems complicated to rotate the rendered text since there are no functions given to do so. It could be an issue in the future. Regarding the HUD design, it is pretty simple yet powerful: we need to create an abstract class if the common attributes, and utility functions. Finally we just need to create inheriting classes that implement update() and render() functions.
