    • About

    How has this website been built?

    It has been built from scratch using Bootstrap 4, HTML5 and CSS. It uses Bootstrap JS scripts, so JQuery comes with it, as well as Popper.js. The icons are provided by Font Awesome 5 Free. The nero.dev domain, and the font are provided by Google. Also, the website is hosted by Github pages. Browsers compatibility (tested 26/08/19):

    • Brave optimized
    • Firefox working
    • Chrome working
    • Opera untested (contact me if you can test it :D)
    • Safari untested (contact me if you can test it :D)
    • Edge some issues
    • Internet Explorer not working

    Also it is compatible with phones.

    Can I get the code?

    The github repository is private, and will stay private because there are sensible files. But if you need anything, feel free to ask by sending an email. In any case, if you use some code, you must provide some credits somewhere.

    What about the images?

    All the images have been done by me. If not, you will be provided with credits.
